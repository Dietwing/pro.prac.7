# Задание 7. Интерактивная панель на Plotly Dash

Проект представляет собой веб-приложение для анализа данных о ценах и объемах продаж авокадо в США.

## Состав проекта

- `app.py` - основной файл приложения Dash
- `data/avocado.csv` - набор данных
- `assets/style.css` - стили интерфейса
- `assets/favicon.ico` - иконка приложения
- `requirements.txt` - зависимости Python

## Функциональные возможности

- выбор региона наблюдения
- выбор типа продукции
- задание временного диапазона
- отображение ключевых показателей
- построение графика средней цены
- построение графика объема продаж
- построение диаграммы средней цены по годам

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

После запуска приложение будет доступно по локальному адресу, который выводится в консоли.
